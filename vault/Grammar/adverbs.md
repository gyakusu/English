# Adverbs（副詞）

<!-- Add a new ## section each time you find a mistake -->

## somewhat（程度副詞）— 形容詞・動詞を「やや〜」と和らげる

- Issue: "somewhat constrained" の somewhat が何を修飾しているか分からなかった
- Rule: somewhat は**程度副詞**で、直後の形容詞・副詞・動詞・分詞を修飾し、「ある程度・やや・多少」と意味を和らげる（mitigation / hedging）。先行詞や別の名詞を指す代名詞ではない。
- ✗ 間違えやすい解釈: somewhat が前の文脈中の名詞（予算など）を指す代名詞だと考える
- ✓ 正しい解釈: somewhat は直後の形容詞 constrained を修飾する副詞。"our budget is [somewhat] constrained" → 「予算が**やや**制約されている」
